HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Everything should be organized with code for it to look nice within the body.
  • Include meta tags for the webpage to appear in web searches.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can code CSS within the HTML, but it's better to do it separately.
  • You can apply the same effects to different sections by using a comma so it takes up less coding space.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add the new branch to the current working branch.
  • git commit -m "commit message": commit with a description of what it is.
  • git pull origin main: makes sure the latest version is in sync with the local project.
  • git push origin branch-name: push our local branch changes to GitHub.
